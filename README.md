# MyTaskManager

Приложението **MyTaskManager** е Android приложение за управление на задачи. Потребителите могат да добавят, редактират и изтриват задачи, както и да преглеждат списъка с тях в удобен интерфейс.

## Функционалности

- Добавяне на нови задачи
- Редактиране на съществуващи задачи
- Изтриване на задачи
- Преглед на списъка с всички задачи
- Поддръжка на различни статуси на задачите (например "в процес", "завършено")
- Прост интерфейс с Android Views и/или Jetpack Compose

---

## Технологии

- Kotlin 2.0.21
- Android Gradle Plugin
- KSP (Kotlin Symbol Processing)
- Android Jetpack (ViewModel, LiveData, Room и други)
- JUnit4 за unit тестове
- Espresso за инструментални UI тестове
